# Bongo Cat Autoclicker

Автоматизированный скрипт для сбора сундуков в игре с помощью компьютерного зрения и эмуляции нажатий клавиш.

## 📥 Скачать Zip архив

[![Скачать Zip](https://img.shields.io/badge/Download-Windows-blue?style=for-the-badge&logo=windows)](https://github.com/RoninOnis/BongoCat-auto/releases/tag/BongoCat_auto_v1.0)

## 🚨 Важное ограничение

**При использовании нескольких мониторов скрипт будет работать корректно только если окно игры (BongoCat) находится на основном (главном) мониторе, масштаб интерфейса выставить 2х.** Скрипт осуществляет поиск изображений только в пределах основного дисплея.

## 📋 Функциональность

- 🔍 **Автоматический поиск сундуков** - распознавание сундуков по изображениям (`1.png` и `2.png`)
- ⌨️ **Автонажатие пробела** - когда сундуки не видны на экране
- ⚡ **Горячие клавиши** - удобное управление без интерфейса
- 📊 **Статистика** - отслеживание эффективности работы
- 🎨 **Визуальное оформление** - цветные сообщения в консоли

## 🛠 Установка и настройка

### Требования
- Windows 10/11
- Python 3.7+
- Игра должна быть запущена в оконном режиме

### Установка зависимостей
```bash
pip install pyautogui opencv-python pillow
```

### Настройка изображений
1. Подготовьте скриншоты сундуков в формате PNG
2. Сохраните их как `1.png` и `2.png`
3. Укажите правильные пути к файлам в настройках скрипта:
   ```python
   IMAGEM_BAU = r"D:\1.png"    # Путь к первому изображению
   IMAGEM_BAU1 = r"D:\2.png"   # Путь ко второму изображению
   ```

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `TAB` | Включить/выключить автокликер |
| `SHIFT` | Экстренный выход из скрипта |

## ⚙️ Настройка параметров

Все основные параметры можно настроить в начале файла `main.py`:

```python
# Тайминги нажатий
MIN_PRESS_DUR = 0.04    # Минимальное время нажатия (сек)
MAX_PRESS_DUR = 0.07    # Максимальное время нажатия
MIN_DELAY = 0.01        # Минимальная пауза между кликами
MAX_DELAY = 0.04        # Максимальная пауза

# Горячие клавиши
TOGGLE_HOTKEY = 0x09    # TAB
EXIT_HOTKEY = 0x10      # SHIFT

# Чувствительность распознавания
confidence=0.7          # Точность распознавания изображений
```

## 🔧 Решение проблем

### Скрипт не находит сундуки
1. Убедитесь, что игра запущена на **основном мониторе**
2. Масштаб интерфейса 2х
3. Проверьте, что изображения точно соответствуют сундукам в игре
4. Уменьшите значение `confidence` до 0.6
5. Запустите игру в оконном режиме

### Ошибки с OpenCV
```bash
# Если возникают ошибки с confidence:
pip install opencv-python
```

### Изображения не распознаются
- Используйте скриншоты без сжатия
- Убедитесь, что фон достаточно контрастный
- Попробуйте делать скриншоты при одинаковом освещении в игре
