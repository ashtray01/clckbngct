# 🎹 Keyboard Masher / Клавиатурный кликер

[![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)](https://www.python.org/)
[![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)](https://www.microsoft.com/windows)

## 📝 Описание

Keyboard Masher - это утилита для Windows, которая эмулирует одновременное нажатие всех клавиш клавиатуры с регулируемой скоростью. Программа создает случайные задержки между нажатиями для имитации естественного поведения.

### 🎯 Возможности

- ⚡ Одновременное нажатие ВСЕХ клавиш клавиатуры
- 🎲 Рандомизированные задержки между нажатиями (0.01-0.04 сек)
- ⏱️ Рандомизированная длительность нажатий (0.02-0.04 сек)
- 📊 Подсчет статистики (время работы, количество нажатий, скорость)
- 🎮 Управление через горячие клавиши
- 🔄 Работа в фоновом режиме

## 🚀 Установка

### Требования
- Windows OS
- Python 3.x
- Права администратора не требуются

### Быстрый старт

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/yourusername/keyboard-masher.git
cd keyboard-masher
```

2. **Запустите скрипт:**
```bash
python keyboard_masher.py
```

## 🎮 Использование

### Управление

| Клавиша | Действие |
|---------|----------|
| **TAB** | Включить/Выключить клики |
| **SHIFT** | Выйти из программы |

### Что нажимается:

| Категория | Клавиши |
|-----------|---------|
| **Буквы** | A-Z (все латинские буквы) |
| **Цифры** | 0-9 |
| **Специальные** | Enter, Esc, Backspace, Delete, Insert, Home, End, Page Up, Page Down |
| **Символы** | - = [ ] ; ' , . / \\ ` |
| **Другие** | Пробел |

### Пример работы

```
==================================================
             КЛАВИАТУРНЫЙ КЛИКЕР                 
==================================================
Управление:
  TAB   - Вкл/Выкл клики
  SHIFT - Выход из программы

Нажимаются все клавиши одновременно:
  • Буквы A-Z
  • Цифры 0-9
  • Специальные клавиши (Enter, Esc, Backspace...)
  • Символы (- = [ ] ; ' , . / \ `)
  • Пробел
==================================================
```

## ⚙️ Настройка

Вы можете изменить параметры работы в начале скрипта:

```python
# Настройки кликера
MIN_PRESS_DUR = 0.02  # Минимальное время нажатия (сек)
MAX_PRESS_DUR = 0.04  # Максимальное время нажатия
MIN_DELAY = 0.01      # Минимальная пауза между кликами
MAX_DELAY = 0.04      # Максимальная пауза
TOGGLE_HOTKEY = KEY_TAB  # Клавиша для вкл/выкл
EXIT_HOTKEY = KEY_SHIFT  # Клавиша для выхода
```

## 📊 Статистика

При выходе из программы (SHIFT или Ctrl+C) отображается статистика:

```
========================================
           СТАТИСТИКА                  
========================================
Время работы: 0ч 15м 23с
Всего нажатий: 15423
Скорость: 16.67 нажатий/сек
========================================
```

## 🛠️ Технические детали

- **Платформа:** Только Windows (использует Windows API)
- **Библиотеки:** 
  - `ctypes` - для вызова Windows API
  - `threading` - для фонового отслеживания клавиш
  - `random` - для рандомизации задержек

### Как это работает

1. Программа запускается и ожидает нажатия TAB
2. При активации начинается цикл:
   - Одновременное нажатие ВСЕХ клавиш через Windows API
   - Случайная задержка (удержание клавиш)
   - Одновременное отпускание всех клавиш
   - Пауза перед следующим циклом
3. Отдельный поток отслеживает горячие клавиши
4. Статистика собирается в реальном времени

